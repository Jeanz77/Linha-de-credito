 <h1> <strong> 📌 Sistema para linha de crédito </strong> </h1>

 🔹Este programa tem como objetivo auxiliar setores financeiros na tomada de decisões quanto abertura de linha de crédito, utilizando de importantes atributos para calcular e obter os resultados.
<br> <br>

 <h3> <strong> 🚀 Funcionalidades </strong> </h3>
 
✔️ Nivel de risco - Através de um sistema de pontuação, informa o grau de risco do cliente e se é viável oferecer linha de crédito.

✔️ Crédito sugerido - Analisa e sugere a partir dos dados fornecidos o valor para linha de crédito.

✔️ Interface simplificada - Fácil de compreender e usar.

<br> 

<h3> <strong> 📦 Pré-requisitos </strong> </h3>

Sistema funciona diretamente do navegador a sua escolha através do link: https://linha-de-credito.vercel.app/
